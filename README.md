# 小米手环自定义表盘工具箱

> a.k.a. **Miband Custom Watchface Toolkit** (MCWT)

该项目旨在为小米手环第三方表盘创作者提供触手可及、开箱即用的趁手工具，提升表盘制作中一些常见操作的效率。

## ⚙️ 功能列表

- [x] 字符纹理生成器 - 用于快速地创建表盘上的字符纹理序列，省去一张张用PS导出的麻烦。
  - 自动对齐字符基线
  - 支持字体名称、字体功能、字体样式控制
  - 支持自定义长宽、边距
  - 同时支持导出单字或多字序列
- [ ] 更多功能敬请期待……
